# Dev Log

This folder stores durable handoff notes for continuing work across chats.
The standing protocol lives in `../DEVLOGS.md`.

When creating a full prompt for the next chat:

- inspect the latest entries in this folder first
- summarize the current architecture, changed files, blockers, and next steps
- save the handoff as a dated Markdown file in this folder
- update `../ROADMAP.md` so the roadmap reflects the session's actual progress

Use concise, implementation-facing notes. Do not store secrets.
